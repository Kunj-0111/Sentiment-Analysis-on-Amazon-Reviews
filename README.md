# Sentiment-Analysis-on-Amazon-Reviews
NLP-based sentiment analysis project on Amazon product reviews using Python and TextBlob.

# Amazon Review Sentiment Analysis

## 📌 Project Overview
This project performs sentiment analysis on Amazon product reviews using Natural Language Processing (NLP) techniques.

The objective is to classify customer reviews into positive, negative, and neutral sentiments and generate business insights from customer feedback.

---

## 🚀 Features
- Data Cleaning
- Sentiment Analysis using TextBlob
- Polarity Score Calculation
- Sentiment Classification
- Data Visualization
- Business Insights

---

## 🛠️ Technologies Used
- Python
- Pandas
- TextBlob
- Matplotlib
- Seaborn

---

## 📊 Key Insights
- Majority of reviews were positive
- Negative reviews highlighted product quality issues
- Sentiment analysis helps businesses understand customer satisfaction
